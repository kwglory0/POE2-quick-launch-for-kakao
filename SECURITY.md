# 보안 정책 (Security Policy)

프로젝트를 보호하고 사용자들에게 안전한 환경을 제공하기 위한 보안 가이드라인입니다.

## 지원되는 버전

현재 다음 버전에 대해서만 보안 업데이트가 지원됩니다.

| 버전     | 지원 여부     |
| :------- | :------------ |
| >= 1.2.0 | ✅ 지원됨     |
| < 1.2.0  | ❌ 지원 안 함 |

## 보안 취약점 제보 방법

본 프로젝트에서 보안 취약점을 발견하신 경우, 다음 절차를 따라주시기 바랍니다.

1.  GitHub의 **공개 이슈(Issue)를 생성하지 마십시오.** 취약점이 악용될 수 있습니다.
2.  취약점에 대한 상세 설명과 재현 방법을 작성하여 프로젝트 관리자에게 제보하여 주십시오.
    - 제보 채널: [GitHub Private Vulnerability Reporting](https://github.com/NERDHEAD-lab/POE2-quick-launch-for-kakao/security/advisories/new) 또는 관리자 이메일 (`nerdlab.kr@gmail.com`)
3.  제보를 받은 후 영업일 기준 3일 이내에 초기 응답을 드릴 수 있도록 노력하겠습니다.

항상 프로젝트의 보안에 기여해 주셔서 감사합니다.
