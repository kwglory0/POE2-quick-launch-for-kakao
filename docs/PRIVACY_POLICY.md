# 개인정보 처리방침 (Privacy Policy)

**POE2 Quick Launch for Kakao** (이하 "본 확장 프로그램")는 사용자의 개인정보를 매우 중요하게 생각하며, 다음과 같은 원칙을 준수합니다.

## 1. 개인정보 수집 및 이용
본 확장 프로그램은 **사용자의 개인정보를 서버로 전송하거나 수집하지 않습니다.**
모든 데이터 처리는 사용자의 브라우저 내에서 로컬(Local)로 이루어지며, 외부 서버로 유출되지 않습니다.

## 2. 데이터 저장 (Local Storage)
본 확장 프로그램은 편의 기능을 제공하기 위해 다음 정보를 사용자의 브라우저 로컬 스토리지(`chrome.storage.local`)에 저장합니다.
- **설정 값**: 자동 탭 닫기, 팝업 닫기, 플러그인 비활성화 여부 등
- **캐시 데이터**: 최근 확인한 패치노트 목록 (로딩 속도 개선 목적)

이 정보는 오직 사용자의 PC 내에만 저장되며, 확장 프로그램을 삭제하면 함께 제거됩니다.

## 3. 외부 링크
본 확장 프로그램은 게임 공식 홈페이지, 거래소, GitHub 저장소 등 외부 웹사이트로의 링크를 포함하고 있습니다.
해당 외부 사이트의 개인정보 처리방침은 본 확장 프로그램과 무관하므로, 각 사이트의 정책을 참고하시기 바랍니다.

## 4. 문의하기
개인정보 처리와 관련하여 문의가 있으신 경우, [GitHub Issue Tracker](https://github.com/NERDHEAD-lab/POE2-quick-launch-for-kakao/issues)를 통해 문의해 주시기 바랍니다.
